PageMonitor
===========

with Mandrill support

## Install
- `npm install`

## Configure
- rename `config.exampe.js` to `config.js`
- edit `config.js`
- add custom provider to `monitor/providers/`

## Usage
- `node page-monitor.js`

## Usage as service
- `npm install pm2 -g`
- `pm2 start page-monitor.js`
